# Cycle App Case Study

A concise index for the product and technical decisions behind Cycle Companion.

## Summary

Cycle Companion is a privacy-first wellness app for understanding patterns across cycle phase, energy, symptoms, food, workouts, and recovery.

The app avoids fertility prediction, diagnosis, treatment advice, and medical claims.

## Highlights

- Dashboard-first product experience
- Local-first IndexedDB storage
- Cycle phase visualization
- Lightweight daily, food, workout, and recovery logging
- Rule-based insights
- Privacy/settings screen
- Web-first architecture with a future mobile path
