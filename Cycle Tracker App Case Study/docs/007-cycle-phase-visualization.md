# Cycle Phase Visualization

## Decision

Use cycle visualizations as a signature part of the product experience.

## Why This Matters

The app should make cycle phase information easy to understand at a glance. A strong visual system also helps the project stand out in a portfolio.

## Visualization Ideas

- Cycle phase ring
- Horizontal cycle timeline
- Calendar with subtle phase colors
- Current phase card
- Estimated phase range labels
- Symptom markers by cycle day

## Design Notes

- Use the phase palette from the design system.
- Show uncertainty with words like "estimated" or "likely."
- Avoid presenting predictions as medical certainty.
- Keep the visualization readable on both desktop and mobile.

## Tradeoff

The visualization should be beautiful, but it must stay clear and honest about prediction uncertainty.
